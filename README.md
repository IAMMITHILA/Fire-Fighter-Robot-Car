# Fire-Fighting-Robo-Car

Bangladesh is a densely populated country where fire incidents occur frequently. It often causing significant damage to lives and property so, to  address this challenge me and my team **Tech Lightning** developed a **Fire Fighting Robo Car**.




**The Fire-Fighting Robo Car** is an **Andino Uno based robotic System** that designed to automatically detect and extinguish fires. Since there are many locations in Bangladesh where it is difficult or unsafe for firefighters to access therefore we developed a small scale prototype to demonstrate how a robotic system can assist in firefighting operations. Also to automatic fire detection and extinguishing the robo car can be controlled remotely using a mobile device. It also features an onboard camera that provides live monitoring of the surrounding environment which allowing users to observe firefighting operations from a safe distance.


Our team **Tech Lightning** designed and presented this project at the **Project Showcase on Intelligent Perception** at **Bangladesh University** on **23 September, 2023** where we achieved **2nd Runner-Up** among the participating teams.
