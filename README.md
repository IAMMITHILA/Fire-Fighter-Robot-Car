# Fire-Fighting-Robo-Car
An Arduino Uno based fire fighting robo car capable of detecting and extinguishing fires autonomously  with mobile control and live camera monitoring.
An Arduino Uno–based intelligent robotic system designed to automatically detect and extinguish fires in hazardous environments where human access may be difficult or unsafe. The robot can also be controlled remotely using a mobile device, and an onboard camera provides live monitoring of the surrounding area.
